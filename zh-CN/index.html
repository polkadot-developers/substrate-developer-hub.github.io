为创新者打造的区块链开发工具

Substrate 是一个模块化框架，让您可通过制作及整合不同的模块来创建定制化的区块链。

为何用 Substrate?

专注回您的长处

Substrate 模块化的设计代表您只需专注开发与您至关重要的自定义组件，而同时可重用那些历经考验的程序模块。

Runtime 开发

使用 Substrate 的 FRAME runtime 系统来构建安全的、高扩展的区块链逻辑。

Substrate 为何物？

您构建区块链所需的一切。

Substrate 后面有着一流的密码学研究支撑，并具有 p2p 网络，共识机制，及其他额外的功能。

快速高效的数据库

模块化的 P2P 网络技术

热插拔的共识层。

可自定义的交易队列

具弹性的 Runtime 库

已优化的轻客户端

可用于生产环境

Substrate 是 Polkadot 背后的框架，而 Polkadot 则是新一代，丰富多样，支持多链的区块链网络。这生态中的大多数区块链也都建立在 Substrate 之上。 Polkadot 网络则是于 2020 年 5 月正式发布。

支持智能合约

Substrate 自带一个的 Web Assembly (Wasm) 智能合约平台。因为 Substrate 使用 Wasm，您可使用任何兼容的语言构建智能合约。为此我们也开发了一个基于 Rust 的智能合约语言 ink!。
