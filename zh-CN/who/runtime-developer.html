Runtime 开发者

如果你想构建区块链......

  1. 不要做独行侠！

    在开始旅程之前，你需要熟悉那些将会帮助到你的资料。我们提供了基本文档，帮助你理清各种术语，向你展示 Substrate 是什么。我们提供 [substrate] StackOverflow 标签，你可以使用它来提出技术问题或找到现有的答案。最后，我们有一个友好、技术性的聊天室，不管在什么时间，这些开发者都乐于帮助你。

    高级文档StackOverflowRiot Chat

  2. 安装 Substrate

    你需要做的第一件事是在你的电脑上安装 Substrate。安装说明根据你所使用的操作系统会有所不同，看一下这里的指南来找到适合你的安装说明。

    安装说明

  3. Substrate Collectables Workshop

    接下来，跟随我们的 Substrate Collectables Workshop 来深入了解 runtime 开发。我们会带你走进整个过程，包括构建一个不可替代的通证去中心应用链，运行你的应用链，并构建用户界面。

    开始工作坊

  4. 引用文档

    现在你已经熟悉了 Substrate 的 runtime 开发，可以跳到 Substrate 核心代码的引用文档。作为开始，你可以通过搜索“srml”调研我们的 Substrate Runtime Module Library (SRML)。你现在应该都够看到支撑这些模块的代码，通过学习这些代码中常见模式来拓展你的知识。

    参考文档

  5. 构建

    你现在可以开始构建自己的 Runtime 逻辑了！别忘了我们为你准备的社区和文档资源。
