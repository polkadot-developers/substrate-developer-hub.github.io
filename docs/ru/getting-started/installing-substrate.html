Installing Substrate

Чтобы настроить компьютер для создания Substrate, вам нужно установить необходимое программное обеспечение.

Основанные на Unix операционные системы

Инструкции по установке Substrate для Mac OS, Arch или ОС на основе Debian, например Ubuntu

Быстрая установка

Если Вы хотите начать быстро создавать на Substrate, запустите этот простой однострочник:

curl https://getsubstrate.io -sSf | bash -s -- --fast

Эта команда установит все зависимости, необходимые для создания узла Substrate, например: Rust, OpenSSL, CMake, LLVM и другое, необходимое для установки. Также будет установлен скрипт Substrate , упрощающий процесс создания собственных узлов и модулей.

Полная установка

Помимо всех элементов, установленных с помощью быстрой установки, полная установка также установит два двоичных файла Substrate на ваш компьютер:

  • subkey: a command line utility that generates or restores Substrate keys.
  • Substrate node: a copy of the main node included with Substrate, allowing you to easily connect to the Substrate test-net.

Вы можете выполнить полную установку используя флаг --fast:

curl https://getsubstrate.io -sSf | bash

После завершения убедитесь, что узел Substrate установлен правильно, проверив его версию:

$ substrate --version
substrate 2.0.0-6f0d28863-x86_64-macos

Вы можете сделать то же самое с Subkey:

$ subkey --version
subkey 2.0.0

Windows

Если Вы пытаетесь настроить компьютер Windows для сборки Substrate, сделайте следующее:

  1. Скачайте и установите "Build Tools for Visual Studio:"
  • Вы можете получить его по этой ссылке: buildtools.
  • Запустите установочный файл: vs_buildtools.exe.
  • Убедитесь, что компонент "Windows 10 SDK" включен при установке инструментов Visual C++ Build Tools.
  • Перезапустите компьютер.
  1. Установите Rust:

  • Подробные инструкции приведены в книге Rust.

    • Скачайте с: https://www.rust-lang.org/tools/install.

    • Run the installation file: `rustup-init.exe`.
  
    > Note that it should **not** prompt you to install `vs_buildtools` since you did it in step 1.

    • Выберите "Установка по умолчанию."

    • Для запуска Вам нужно задать каталог, содержащий бинарные файлы Cagro (%USERPROFILE%\.cargo\bin) в переменной окружения PATH. Потом приложения будут автоматически настраивать правильную среду окружение, но Вам может понадобиться перезапуск Вашей текущей оболочки.

  1. Выполните эти команды в командной строке (CMD) для установки среды сборки Wasm:

    rustup update nightly
rustup update stable
rustup target add wasm32-unknown-unknown --toolchain nightly

  2. Установите wasm-gc, который используется для уменьшения размера файлов Wasm:

    cargo install --git https://github.com/alexcrichton/wasm-gc --force

  3. Установите LLVM: https://releases.llvm.org/download.html

  4. Установите OpenSSL, который нам будет нужен для vcpkg:

    mkdir C:\Tools
cd C:\Tools
git clone https://github.com/Microsoft/vcpkg.git
cd vcpkg
.\bootstrap-vcpkg.bat
.\vcpkg.exe install openssl:x64-windows-static

  5. Добавьте OpenSSL в системные переменные:

    $env:OPENSSL_DIR = 'C:\Tools\vcpkg\installed\x64-windows-static'
$env:OPENSSL_STATIC = 'Yes'
[System.Environment]::SetEnvironmentVariable('OPENSSL_DIR', $env:OPENSSL_DIR, [System.EnvironmentVariableTarget]::User)
[System.Environment]::SetEnvironmentVariable('OPENSSL_STATIC', $env:OPENSSL_STATIC, [System.EnvironmentVariableTarget]::User)

  6. Наконец, установите cmake: cmake download

Ручная сборка

Если Вы хотите покодить на самом Substrate или вручную построить узел, Вы можете клонировать репозиторий Substrate с помощью:

git clone https://github.com/paritytech/substrate.git

Прежде чем Вы сможете скомпилировать проект, Вам нужно установить всё необходимое для сборки. Вы можете посмотреть на getsubstrate.io, чтобы увидеть все зависимости, необходимые для вашей операционной системы.

Далее Вы можете собрать узел с

# Обновите установленный rust toolchain 
./scripts/init.sh

# Создайте нативный узел
cargo build

Вы также можете собрать subkey с помощью

cargo build -p subkey

Больше информации можно получить в readme проекта.
