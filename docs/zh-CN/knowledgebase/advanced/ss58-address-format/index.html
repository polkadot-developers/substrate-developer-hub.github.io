SS58 地址格式

SS58 是一个简单的地址格式，设计用于基于 Substrate 开发的链。 使用其他地址格式也是没有问题的，但 SS58 是一个可靠的默认项。 它在很大程度上基于比特币的 Base-58-check 格式，并作了一些修改。

基本思路是用一个 base-58 编码值代表一个在 Substrate 链上的特定帐户。 不同的链有不同的识别帐户的方法。 因此，SS58 被设计为可扩展的地址格式。

您可以在 Substrate GitHub Wiki 上找到当前 SS58 地址格式的规范：

https://github.com/paritytech/substrate/wiki/External-Address-Format-(SS58)
