Polkadot-JS

Polkadot-JS 项目是一个围绕 Polkadot 和 Substrate 的工具、接口、和库的集成。

注意 ：虽然该项目以 "Polkadot" 命名，但这些工具、接口、和库完全兼容任何基于 Substrate 的链。

Polkadot-JS API

Polkadot-JS API 是一系列用于与 Polkadot 和 Substrate 节点沟通的接口库。

该 API 允许应用程序开发人员查询节点，并让他们使用 Javascript 与 Polkadot 或其他 Substrate 链交互。

文档 GitHub

开始

Polkadot-JS API 的开发者文档可在其网站中找到。 Follow the Getting Started guide to learn how to install and start using the Polkadot-JS API right away.

与本地节点连接

要将 Polkadot-JS Apps 连接到你的本地节点，你须进入 设置，并将 "连接的端点" 更改为 本地节点(127.0.0.1:9944)。

注意：如果你通过安全的 HTTPS 连接连接到 Polkadot-JS 应用程序，你将需要使用同样支持连接到不安全的 WebSocket 端点的浏览器。 例如，Google Chrome 浏览器支持此功能，但 Mozilla Firefox 不支持。

Polkadot-JS 扩展插件

Polkadot-JS 扩展插件是一个简单的概念验证用于管理浏览器扩展里的钱包账户，并允许使用这些账户签署交易。 它还提供了简单的接口用于与扩展兼容的 dApps 进行交互。
