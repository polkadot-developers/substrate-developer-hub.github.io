This page will answer a number of common questions you may have when starting to build smart contracts with Substrate.

ink! F.A.Q.

内存和存储有什么区别？

在 ink! 中， memory 指的是程序员常说的计算机内存，而 storage 指的是合约实例所占用的磁盘空间。 The storage is backed up by the runtime in a database. 我们认为其访问速度较为缓慢。

如何运行测试？
