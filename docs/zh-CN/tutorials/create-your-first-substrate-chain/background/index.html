背景资料

While we wait for the node template to build, let's learn a little bit about blockchains and the Substrate blockchain development framework. The Node Template that you are compiling was built with this framework.

区块链背景知识

区块链网络是由独立的 节点 组成的，节点间通过点对点 (P2P) 网络连接。 节点就是在网络里运行着区块链软件的独立计算机，它们是使得区块链网络正常运作的关键。

一个区块链节点必须满足以下几点才能正常运作：

  • Storage
  • Peer-to-peer networking
  • Consensus capabilities
  • Extrinsic handling (more than just transactions)
  • A Runtime
  • and more...

这些技术横跨计算机科学的多个领域，因此区块链项目一般需要一整个专家级团队来开发。 因而，大多数的区块链项目 并不是 从没到有的开发。 相反，这些项目是从已有的区块链代码库中分叉而来。 例如：

  • 基于比特币代码库而分叉：Litecoin、ZCash、Namecoin、Bitcoin Cash、等等...
  • 基于以太坊代码库而分叉：Quorum、POA 网络、KodakCoin、Musicoin、等等...

以这种方式构建区块链都有严重的局限性，因为这些现存的平台在设计之初并没有考虑到迭代问题。

Substrate

Substrate 是一套 开源、模块化、和 可扩展 的区块链开发框架。

Substrate 是从零开始设计以确保灵活性，使创新者能够设计和构建一个满足其需要的区块链网络。 它提供了构建定制化区块链节点所需要的全部核心组件。

Substrate 开发者中心 Node Template

我们以 Node Template 的方式提供了一个基于 Substrate 的开箱即用的节点，您可一边编译该节点一边阅读本文档。 您和您朋友无需任何的修改即可使用并创建出一个内置加密货币，并可以包含其他功能的可用的区块链网络。

我们将告诉您如何在 “开发” 模式中运行此节点，它让您运行一个单节点网络，并且里面拥有一些预先配好资金的用户账号。
