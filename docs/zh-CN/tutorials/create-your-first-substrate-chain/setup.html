配置您的计算机

通常，我们会教你更多关于 Substrate 区块链开发框架。然而，配置您的计算机作 Substrate 开发也会花费一些时间。

为节约您的时间，我们会先开始安装过程。 在编译过程中，你可以继续到下一节学习更多关于 Substrate 的知识和我们所要开发的东西。

先決条件

为了配置 Substrate 开发环境，您需要做一些准备工作。

Substrate 开发

请参阅 官方安装指南。 由于本教程使用的 Node Template 它包含了用于初始化的配置文件和脚本与该节点进行交互，因此无需完成安装指南中的所有步骤。 安装和配置 rustup ，并且将 Rust toolchain 默认配置为最新的稳定版本后，您可以返回到以下步骤。

编译 Substrate

上面的先决条件满足后，您可使用 Git 来克隆 Substrate 开发者中心的 Node Template。这将为你在 Substrate 开发提供良好的起点。

  1. 克隆 Node Template (v2.0.0)

    git clone -b v2.0.0 --depth 1 https://github.com/substrate-developer-hub/substrate-node-template

  2. 初始化您的 WebAssembly 构建环境

    make init

  3. 编译 Node Template

    make build

编译所需时间取决于您使用的硬件。 不要等待，我们继续吧！

安装 Front-End Template

本教程使用 ReactJS Front-end Template，允许您与 Substrate 区块链节点交互，就是您在前一步已经开始编译的项目。 日后您可以使用这个 Front-End Template 为自己的项目创建介面。

为了使用 Front-End Template，您需要安装 Yarn，这也需要您同时安装 Node.js。 如果您没有这些工具，可按下边的说明来安装：

  • 安装 Node.js
  • 安装 Yarn

现在您可以使用以下命令来设置 Front-End Template。

# 从 github 上克隆代码
git clone -b v2.0.0 --depth 1 https://github.com/substrate-developer-hub/substrate-front-end-template

# 安装依赖
cd substrate-front-end-template
yarn install
