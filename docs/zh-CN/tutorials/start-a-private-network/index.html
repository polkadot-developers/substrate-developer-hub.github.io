启动私有网络

在本教程中，我们将学习和实践如何使用 Substrate 来启动一个拥有你选择的验证节点集的区块链。

安装 Node Template

You should already have version latest of the Substrate Node Template compiled on your computer from when you completed the Create Your First Substrate Chain Tutorial. 如果您尚未完成，请完成该教程。

有经验的开发者可以跳过该教程，您可以参考 readme 里的说明步骤来安装 Node Template 。

你接下来会做的事

在我们开始之前，让我们解释下我们将要在本教程中做些什么。 我们将：

  1. 基于模板项目启动一个 Substrate 区块链网络。
  2. 生成 ed25519 和 sr25519 密钥对，作为网络授权使用。
  3. 创建和编辑 chainspec 的 json 文件。

听起来还不错吧？ 那么，让我们开始吧！
