介绍

In this tutorial, you will learn to create a custom "Proof of Existence" dApp using the Substrate blockchain development framework and the FRAME library.

完成本教程大概需要花费你 1 个小时的时间。 我们将使用 Rust 和 ReactJS，即使你不太了解这些编程知识，依旧能够完成该指南。 我们将为你提供可运行的代码片段，并概略解释这些代码都做了些什么。

我们只期望：

  • You have completed the Create Your First Substrate Chain Tutorial.
  • You are generally familiar with software development, writing code, and running your code.
  • 你对学习区块链开发的前沿技术抱开放态度。

If you run into an issue on this tutorial, we are here to help! You can ask a question on Stack Overflow and use the substrate tag or ask your questions directly in the Substrate Technical Channel.

你接下来会做的事

在我们开始之前，先来看看我们在这个教程中做些什么吧。 我们会一步步地：

  1. 基于节点模板启动一个 Substrate 区块链。
  2. Modify the node template to add our own custom PoE pallet from scratch and implement our PoE API.
  3. 修改前端模板，添加一个 自定义用户界面 与我们的 PoE API交互。

听起来还不错吧？ 那么，让我们开始吧！
